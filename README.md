# Ottawa Agent Systems Website Preview Demos

Public static GitHub Pages demos for Ottawa Agent Systems.

These demos show what a cleaner website or lead-capture page could look like for a local service business. All business details are fictional and demo-safe.

## Included Demos

- `auto-repair-demo/` - Ottawa Auto Care Demo, a fictional auto repair website concept.
- `auto-repair-demo-slate/` - Ottawa Auto Care Slate Demo, a fictional Photo 2 style comparison concept.
- `contra-proof-lab-auto-demo/` - OAS Demo Auto Care, a fictional lead-capture landing page with mocked follow-up workflow.

## Build Mode

- Static HTML, CSS, and JavaScript only.
- No backend.
- No database.
- No analytics or tracking.
- No cookies.
- No secrets, API keys, or paid dependencies.
- No third-party form submission.

## Publishing

This repo is intended for GitHub Pages from the `main` branch root.

If Pages is enabled, the site should be available at:

`https://bobkolemanas-hue.github.io/oas-website-preview-demos/`

## Demo Disclaimer

This is a fictional demo website collection created to show website and lead-capture preview concepts. It is not a real business directory, and the demos do not collect form submissions.
